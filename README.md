Uma solução robusta para o gerenciamento e processamento de transações financeiras digitais, focada em segurança, escalabilidade e performance.

## Sobre o Projeto
Este sistema foi desenvolvido para lidar com o fluxo completo de movimentações financeiras, permitindo o registro, consulta e conciliação de transações entre contas.

### Principais Funcionalidades
✅ Registro de transações (Débito/Crédito)
✅ Histórico detalhado por período
✅ Cálculo de saldo em tempo real
✅ Validação de segurança contra fraudes
